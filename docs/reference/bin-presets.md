# Bin Presets

Quick-add bins with common configurations. Click to enter placement mode, or drag directly onto the grid.

## Available Presets

| Name | Size | Features | Use Case |
|------|------|----------|----------|
| **Small Parts** | 1x1x3u | Stacking lip | Screws, nuts, small components |
| **Screwdriver** | 1x4x6u | — | Long tools, screwdrivers |
| **Socket Tray** | 2x2x2u | 3x3 dividers | Socket sets, small tools |
| **Deep Bin** | 2x2x6u | Stacking lip + label shelf | Bulky parts, tall items |
| **Wide Shallow** | 3x2x2u | Stacking lip | Wide flat items |
| **SFP Tray** | 2x1x2u | 5 X-dividers | SFP modules, network gear |
| **Cable Mgmt** | 1x6x3u | — | Cable runs, long items |

## Adding Bins

### Placement Mode (Click)

1. Click a preset button in the sidebar
2. A ghost preview follows your cursor on the grid
3. Press `R` to rotate before placing
4. Click a valid (green) position to place
5. Press `Esc` to cancel

### Quick Add (+ Button)

Click the **+** button next to a preset to instantly place a bin in the first available position on the grid (top-left, scanning left-to-right, top-to-bottom).

### Drag & Drop

Preset buttons are draggable — drag them directly from the sidebar onto the grid. A drop ghost shows validity as you hover.

## Custom Bins

Place a **1x1 bin** using "+ Place 1x1 Bin", then resize using the edge handles or configure dimensions in the Bin Configurator panel.
