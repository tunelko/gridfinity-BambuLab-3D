# Keyboard Shortcuts

## Selection

| Shortcut | Action |
|----------|--------|
| `Click` | Select a single bin |
| `Shift + Click` | Toggle bin in/out of multi-selection |
| `Ctrl + A` | Select all bins |
| `Click` empty space | Deselect all |

## Editing

| Shortcut | Action |
|----------|--------|
| `R` | Rotate selected bin (swap W and D) |
| `Delete` / `Backspace` | Remove all selected bins |
| `Ctrl + C` | Copy selected bin(s) to clipboard |
| `Ctrl + V` | Paste (auto-place in first free position) |
| `Ctrl + D` | Duplicate selected bin(s) |

## History

| Shortcut | Action |
|----------|--------|
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |

## Navigation

| Input | Action |
|-------|--------|
| `Middle mouse button` | Pan the grid |
| `Scroll wheel` | Zoom (0.2x to 5x) |

## Placement Mode

| Shortcut | Action |
|----------|--------|
| `R` | Rotate the placement ghost |
| `Escape` | Cancel placing / dragging / resizing |

::: info
`R` (rotate) only works with single selection or in placement mode. Resize handles, divider controls, and the rotate handle are only available when a single bin is selected.
:::
