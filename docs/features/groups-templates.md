# Groups & Templates

## Bin Groups

Organize your bins by assigning them to **color-coded groups**. Groups help you visually distinguish bins by category and collapse/expand sections in the sidebar.

### Available Groups

| Group | Color |
|-------|-------|
| Screws | Orange |
| Tools | Blue |
| Cables | Green |
| Electronics | Purple |
| Parts | Cyan |
| Office | Pink |
| Network | Teal |

### Assigning Groups

**Single bin:** Select a bin and use the GROUP section in the Bin Configurator.

**Multiple bins:** Shift+Click to multi-select, then use the bulk GROUP panel.

### Sidebar Organization

When bins belong to multiple groups, the sidebar bin list automatically organizes them into **collapsible sub-sections** per group. Ungrouped bins appear at the bottom. Each group shows a colored dot indicator and a bin count.

## Layout Templates

Load predefined complete layouts to get started quickly.

### Available Templates

| Template | Grid | Description |
|----------|------|-------------|
| IKEA Alex Drawer | 7 x 14 | Drawer organizer with mixed bin sizes |
| Electronics Bench | 6 x 6 | Components, tools, and cable management |
| Tool Wall | 8 x 4 | Wall-mounted tool storage |
| Server Rack 1U | 10 x 8 | Network equipment and SFP trays |

### Loading a Template

1. Open the **TEMPLATES** section in the sidebar
2. Click "Browse Templates"
3. Click any template to load it

::: warning
Loading a template **replaces** your current layout. Save your work first if needed.
:::
